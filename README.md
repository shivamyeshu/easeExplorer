# EaseExplorer - AI-Powered Trip Planning App

### Final Commit

Welcome to the final version of **EaseExplorer**, your personal travel assistant powered by AI, offering seamless travel planning with smart itineraries, curated hotels, and engaging video content.

## Project Overview

EaseExplorer helps users plan trips by:
- Generating custom itineraries based on user preferences.
- Offering personalized hotel recommendations within their budget.
- Featuring local highlights and attractions through videos and reels.

### Key Features

1. **Personalized Itinerary Generator**:
   - AI-powered itineraries based on user interests, travel dates, and budget.
   - Detailed schedules for each day, including travel times and local hotspots.

2. **Hotel Recommendations**:
   - Curated list of hotels with price ranges, locations, and ratings.
   - Visual content including hotel images and maps for convenience.

3. **Engaging Media Content**:
   - Integrated videos and reels for destination previews, local attractions, and travel inspiration.

4. **User-Friendly Interface**:
   - Clean, intuitive UI powered by Tailwind CSS.
   - Responsive design for seamless use across devices.

## Tech Stack

- **Frontend**: React.js,   JavaScript
- **UI**: Tailwind CSS
- **Backend**: Firebase for data storage, real-time database for user preferences and itineraries.
- **APIs**: Google Maps API for geo-coordinates and travel time data.
- **Deployment**: Vercel
